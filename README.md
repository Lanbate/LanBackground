### 如何使用？
 :fa-check: 安装vuecli脚手架
`npm install -g @vue/cli`

 :fa-check: 安装依赖
`npm inst`

 :fa-check: 连接数据库
`config\kes.js\mongoURI里输入mongodb数据库链接`

 :fa-power-off: 项目启动
`npm run dev`


 :fa-flag: 打包
`npm run build`
