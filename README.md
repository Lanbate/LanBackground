### 如何使用？
:heavy_check_mark: 安装vuecli脚手架
`npm install -g @vue/cli`

:heavy_check_mark: 安装依赖
`npm inst`

:heavy_check_mark: 连接数据库
`config\kes.js\mongoURI里输入mongodb数据库链接`

:heavy_check_mark: 项目启动
`npm run dev`

:red_circle: 打包
`npm run build`
