### 如何使用？
 **
#安装vuecli脚手架** 
`npm install -g @vue/cli`

#安装依赖
`npm inst`

#连接数据库
`config\kes.js\mongoURI里输入mongodb数据库链接`

# 项目启动
`npm run dev`


# 打包
`npm run build`
